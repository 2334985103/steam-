# Steam用户资料查询工具  
**功能**  
通过Steam ID查询用户公开资料，包括：头像、昵称、在线状态、游戏成就等基础信息。  

## 环境要求  
- PHP 7.4+  
- Web服务器（Apache/Nginx）  
- HTML5浏览器  

## 快速部署 
1. **上传源码**
   上传源码到你的服务器或者虚拟主机然后解压即可
   
3. **获取Steam API Key**  
   前往 [Steam Web API](https://steamcommunity.com/dev/apikey) 注册免费API密钥
   
5. **配置文件设置**  
   编辑 `steam_query_v5.php` 填入你的API密钥
